## 📖 参考理念

### Rachel Pollack《78度的智慧》
**核心原则：**
1. **符号学深度** - 每个符号的多层意义
2. **荣格心理原型** - 集体无意识的象征
3. **内心动力转化** - 符号→个人心理动能
4. **叙事整合** - 将牌面故事化为生命旅程

**应用方式：**
- 正位：展现原型的整合状态
- 解读重点：内在力量的觉醒与运用
- 避免：外在预测，强调内在转化

### Mary K. Greer《塔罗逆位精解》
**核心原则：**
1. **能量流转** - 逆位是能量的另一种表达
2. **非宿命论** - 逆位≠坏事，是调整信号
3. **多维度解读** - 内化、延迟、过度、缺乏等
4. **赋权视角** - 逆位显示可改变的能量

**应用方式：**
- 逆位：能量内化/阻滞/过度/学习中
- 解读重点：如何调整能量流动
- 避免：负面预言、宿命论

## 🎯 升级策略

### 新的解读结构

**正位（Upright）：**
```
1. 原型象征 (Archetype)
   - 荣格原型意义
   - 符号的深层含义
   
2. 内在动力 (Inner Dynamic)
   - 这股力量在你内心的显现
   - 如何与这个原型合作
   
3. 整合建议 (Integration)
   - 具体的心理整合步骤
   - 日常生活中的体现
```

**逆位（Reversed）：**
```
1. 能量状态 (Energy Pattern)
   - 能量的当前流动方式
   - 是内化/阻滞/过度/学习
   
2. 调整方向 (Adjustment)
   - 如何重新校准能量
   - 需要什么觉察
   
3. 成长机会 (Growth Edge)
   - 这个逆位带来的礼物
   - 深化理解的机会
```
