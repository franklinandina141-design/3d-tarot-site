## 🎯 核心改进原则

根据你的要求，已实现以下改进：

### ✅ 1. 禁止术语堆砌
**之前：**
```
"偉特牌義提示：內在對齊。這是一張偏向「順流但需要覺察」的訊號。"
```

**现在：**
```
"在工作上，你可能正经历：明明很努力，但成果不如预期..."
```

### ✅ 2. 完全场景化
**事业问题示例：**
```javascript
if (questionType === '事业') {
  depth += `在工作中，你可能正经历这样的状态：\n`;
  depth += `- 明明很努力，但成果不如预期\n`;
  depth += `- 开始怀疑自己的选择，或者对工作失去热情\n`;
  depth += `- 感觉被困在某个位置上，想动却不知道往哪动\n\n`;
}
```

**感情问题示例：**
```javascript
if (questionType === '感情') {
  depth += `在感情里，你可能感觉到：\n`;
  depth += `- 说不清的不对劲，但又说不出具体问题\n`;
  depth += `- 想亲近对方，又莫名想保持距离\n`;
}
```

### ✅ 3. 讲故事模式
```javascript
story += `在此之前（${past.name}），`;
story += `你经历了一段需要向内调整的时期...\n\n`;
story += `但最近（${now.name}），`;
story += `事情开始放缓了。不是说倒退，而是你的内心在说：\"等一等..."\n\n`;
story += `接下来（${future.name}），`;
```

### ✅ 4. 逆位是提醒，不是障碍
```javascript
// 避免使用"失败"、"坏事"
depth += `但这不是失败，而是你的内心在说："等一等，我需要重新确认方向。"\n\n`;

// 温柔的提醒
climate += `这两张逆位牌像两个温柔的路标，在告诉你："慢一点，这里需要你停下来看仔细。"`;
```

### ✅ 5. 具体行动
**事业类：**
```javascript
guidance.action = `**今天就可以做的**：\n`;
guidance.action += `1. 今晚花30分钟，写下"我理想中的工作是什么样的"（不要想可不可能，先写出来）\n`;
guidance.action += `2. 明天找一个你信任的朋友或前辈，聊聊你的困惑\n`;
guidance.action += `3. 这周末给自己一个下午的"空白时间"——不安排任何任务，就是休息`;
```

**感情类：**
```javascript
guidance.action = `1. 今晚写一封"不寄出的信"给对方，说出所有你不敢说的话\n`;
guidance.action += `2. 这周找一个安静的时间，跟对方说："我想聊聊我们的关系。"\n`;
guidance.action += `3. 每天留出15分钟独处时间，问自己："在这段关系里，我真正想要的是什么？"`;
```

### ✅ 6. 语气温暖
```javascript
// 共情开场
climate = `我看到你的三张牌都是逆位——这通常意味着，你现在处在一个需要"向内看"的阶段。`;

// 鼓励结尾
closing += `深呼吸，感受一下——你的直觉在说什么？相信它。你比你以为的更有智慧。`;
```

## 📝 具体示例对比

### 场景1：事业问题 + 逆位牌

**之前（书面化）：**
```
"這張逆位的大阿卡納正在呼喚你進行一場內在的煉金術。
這種轉化可以描述為從「束縛」到「解放」的轉化。"
```

**现在（场景化）：**
```
"在工作中，你可能正经历这样的状态：
- 明明很努力，但成果不如预期
- 开始怀疑自己的选择，或者对工作失去热情
- 感觉被困在某个位置上，想动却不知道往哪动

但这不是失败，而是你的内心在说："等一等，我需要重新确认方向。"

具体行动：这周找个下午，列出"我真正想要的工作是什么样的"——
不是应该要，是真正想要。写下来后，对照现状，看看差距在哪。"
```

### 场景2：感情问题 + 正位牌

**之前（抽象）：**
```
"這張正位的牌顯示，能量在這個領域相對暢通。
但「正位」不等於「完美」——它只是代表能量在流動。"
```

**现在（具体）：**
```
"在感情里，能量流动意味着：
- 你们的互动比较自然
- 有共同的节奏
- 或者至少，问题比较清晰

但要记住：顺畅不等于可以不努力。

具体行动：这周找一个晚上，关掉手机，跟对方一起做顿饭（或者散步）。
重点不是做什么，是专注地相处。"
```

## 🎬 故事化叙述示例

**过去→现在→未来的故事线：**

```
你的故事是这样走到这里的：

在此之前（愚者），你可能经历了一段相对自由、充满可能性的时期。
那时候你对很多事情都充满好奇，也愿意尝试。

但最近（恶魔·逆位），你开始感觉被某种模式困住了。
可能是工作上的重复、感情里的循环、或者生活方式的固化。
这让你有点焦虑，想改变又不知道怎么开始。

接下来（星星），好消息是能量会开始向外流动。
只要你愿意在现在这个阶段诚实面对自己的感受，
未来会自然打开。关键在于：不要急。
给自己一到两周的时间，让内在整合完成。
```
