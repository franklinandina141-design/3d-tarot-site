# 📷 视频添加小头像贴图教程

## 🎯 两种方法

### 方法1：用剪映添加头像（最简单）⭐⭐⭐⭐⭐

**步骤：**

#### 1. 准备头像素材
```
拍一张照片 或 用现有照片
建议：
- 正方形（1:1）
- 清晰的脸部特写
- 纯色背景（方便抠图）
```

#### 2. 在剪映中导入视频
```
1. 打开剪映
2. 导入你剪好的视频
```

#### 3. 添加头像（画中画）
```
1. 点击「画中画」→「新增画中画」
2. 选择你的头像照片
3. 调整大小和位置
```

#### 4. 去除背景（抠图）
```
1. 选中头像图层
2. 点击「智能抠图」
3. 选择「人像抠图」
4. AI自动去除背景
5. 完成！头像变透明了
```

#### 5. 调整头像位置
```
常见位置：
- 右下角（最常见）
- 左下角
- 右上角

大小建议：
- 占画面的 1/8 到 1/6
- 不要太大（抢镜）
- 不要太小（看不清）
```

#### 6. 添加边框（可选）
```
1. 选中头像
2. 点击「描边」
3. 选择白色或浅色
4. 宽度：3-5px
5. 让头像更突出
```

#### 7. 添加动画（可选）
```
入场动画：
- 「放大弹入」（活泼）
- 「淡入」（柔和）
- 「从下弹入」（自然）

全程动画：
- 「轻微摇摆」（可爱）
- 「呼吸效果」（生动）
```

## 🎨 头像素材准备

### 拍摄技巧

**光线：**
- 明亮、均匀
- 避免强烈阴影
- 自然光最好

**背景：**
- 纯色（白色/灰色/浅色）
- 避免杂乱
- 方便后期抠图

**表情：**
- 微笑（亲和力）
- 自然（不要太夸张）
- 眼神看向镜头

**角度：**
- 正面或3/4侧面
- 稍微仰拍（显脸小）
- 头部居中

## 📋 剪映操作步骤（详细版）

### Mac版剪映

```
1. 打开剪映专业版
2. 导入视频到时间轴
3. 点击顶部「画中画」
4. 点击「新增画中画」
5. 选择头像照片
6. 照片出现在画面上
7. 点击「智能抠图」（右侧面板）
8. 选择「人像抠图」
9. 等待3-5秒，AI自动去背景
10. 拖动头像到右下角
11. 调整大小（双指缩放或拖动边角）
12. （可选）添加「描边」：
    - 颜色：白色
    - 宽度：4px
13. （可选）添加动画：
    - 入场：放大弹入
    - 出场：缩小消失
14. 播放预览
15. 满意后导出
```

### 手机版剪映

```
1. 打开剪映App
2. 点击「开始创作」
3. 选择你的视频
4. 点击底部「贴纸」
5. 点击「自定义贴纸」
6. 选择你的头像照片
7. 自动抠图（AI）
8. 拖到右下角
9. 双指缩放调整大小
10. 点击「样式」添加边框
11. 完成！
```
