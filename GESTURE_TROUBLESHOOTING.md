## 🔍 常见问题排查

### 问题1：手势完全不响应

**可能原因：**
- 摄像头权限未授予
- 摄像头被其他应用占用
- 浏览器不支持MediaPipe

**解决方案：**
```
1. 检查摄像头权限（浏览器地址栏左侧的图标）
2. 关闭其他使用摄像头的应用（Zoom、Teams等）
3. 尝试Chrome/Edge浏览器（Safari可能不稳定）
```

### 问题2：手势识别延迟或卡顿

**可能原因：**
- CPU占用过高
- 数据文件加载过大
- RAF动画循环问题

**解决方案：**
```
1. 清除浏览器缓存 (Cmd+Shift+R)
2. 关闭其他标签页
3. 检查CPU使用率（活动监视器）
```

### 问题3：手势识别不准确

**可能原因：**
- 光线太暗
- 手离摄像头太远/太近
- 手势动作太快

**解决方案：**
```
1. 调整光线（明亮环境）
2. 手离摄像头约30-50cm
3. 放慢手势动作
4. 确保手掌完全在画面内
```

---

## 🎯 推荐的手势操作

### ✅ 正确的手势
- **张开手掌** - 五指分开，掌心朝向摄像头
- **保持稳定** - 手势动作缓慢、稳定
- **适当距离** - 手离摄像头30-50cm

### ❌ 避免的动作
- **手势过快** - 太快会导致检测失败
- **手指遮挡** - 手指重叠会影响识别
- **背景复杂** - 背景有手形物体会干扰
