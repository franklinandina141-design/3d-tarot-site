## 🚀 快速部署

### 部署 Claude（推荐）

```bash
1. 访问 https://console.anthropic.com/ 获取 API Key
2. 使用 worker-claude.js 部署
3. 环境变量设为 ANTHROPIC_API_KEY
4. 完成！
```

**详细步骤：** 查看 `DEPLOY-CLAUDE.md`

### 部署 GPT-4o/mini

```bash
1. 访问 https://platform.openai.com/api-keys 获取 API Key
2. 使用 worker.js 部署
3. 环境变量设为 OPENAI_API_KEY
4. (可选) 修改 model 为 gpt-4o-mini
5. 完成！
```

**详细步骤：** 查看 `DEPLOY.md`
