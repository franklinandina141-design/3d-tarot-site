# 🎥 录制指南 - 一步一步来

## 📋 录制前准备

### 1. 清理桌面和浏览器
```bash
# 关闭不需要的标签页
# 清理桌面图标（录屏时会看到）
# 关闭通知（勿扰模式）
```

### 2. 打开网站
```
http://localhost:8000/index.html
```

### 3. 准备一个测试问题
```
建议问题：「这段关系还值得投入吗？」
（情感类问题最能展示深度解读效果）
```

---

## 🎬 开始录制！

### 第一步：启动系统录屏

**按键：** `Cmd + Shift + 5`

**设置：**
1. 选择「录制整个屏幕」（或「录制选定部分」如果只想录浏览器）
2. 点击「选项」：
   - 存储位置：桌面（方便找到）
   - 麦克风：关闭（不需要旁白，后期加字幕）
   - 显示鼠标点击：开启（更清晰）
3. 点击「录制」按钮

---

## 🎭 录制流程（跟着做即可）

### 场景1：开场（10秒）
```
动作：
1. 保持网站首页静止 3 秒
2. 慢慢移动鼠标到标题上悬停
3. 移动到输入框
```

### 场景2：输入问题（10秒）
```
动作：
1. 点击输入框
2. 慢慢打出：「这段关系还值得投入吗？」
   （不用太快，让观众看清）
3. 鼠标移动到「开啟靈魂儀式」按钮
4. 悬停 1 秒
5. 点击
```

### 场景3：抽牌过程（20秒）
```
动作：
1. 等待进入抽牌页面
2. 如果手势识别正常工作：
   - 挥动手掌选牌
   - 捏合手指确认（重复3次）
   
3. 如果手势识别不稳定（没关系）：
   - 直接点击卡牌（效果也很好）
   
4. 等待三张牌全部选完
5. 等待自动跳转到结果页面
```

### 场景4：解读结果（30秒）
```
动作：
1. 到达结果页面后，停留 3 秒
2. 开始慢慢向下滚动
3. 在每个板块停留 3-5 秒：
   
   - 「靈魂氣候總覽」 → 停 3 秒
   - 「現狀深度解碼」 → 停 5 秒（重点）
   - 「你的故事如何走到這裡」 → 停 3 秒
   - 「你現在最需要的方向」 → 停 5 秒（重点）
   - 「給你的溫柔結語」 → 停 5 秒（重点）
   
4. 滚动到底部后停留 3 秒
```

### 场景5：结束（5秒）
```
动作：
1. 慢慢滚动回顶部
2. 停留在三张牌的位置
3. 保持静止 3 秒
```

## ⏸️ 停止录制

**按键：** `Cmd + Ctrl + Esc`  
或点击菜单栏的停止按钮

**视频会自动保存到桌面！**

## 💡 录制技巧

### ✅ 要做的：
- 鼠标移动要慢而流畅
- 每个关键动作后停留 1-2 秒
- 保持节奏平稳

### ❌ 不要担心：
- 录长了没关系（我会剪掉多余部分）
- 录错了可以重录（很快）
- 手势识别不工作就用点击（完全没问题）
- 不需要完美（后期处理会很流畅）

---

**录制时间：** 2-3 分钟
