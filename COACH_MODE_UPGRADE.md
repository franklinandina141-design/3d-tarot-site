## 📋 新增功能

### 1. 当下局势的真相
**特点:**
- ✅ 一针见血，直接点破核心困境
- ✅ 使用心理学术语（认知失调、冒名顶替综合症、情感投射）
- ✅ 禁止模棱两可，不说"可能""或许"
- ✅ 直接回应问题关键词

**示例输出:**
```
关于「你的项目」，你现在卡在一个认知失调的死循环里：
你的理性告诉你应该前进，但你的情感却在收缩。
这种撕裂感会让你陷入「行动瘫痪」——既不敢前进，也不愿后退。
```

### 2. 隐藏的阻碍与冲突
**特点:**
- ✅ 分析牌与牌之间的能量冲突
- ✅ 点出心理盲点
- ✅ 暴露自我欺骗模式

**示例输出:**
```
你试图从【愚者】的fire能量，切换到【隐士】的water状态。
但这两种能量在本质上是互斥的——就像你一边踩油门一边踩刹车。

你最大的盲点是「选择性注意」——你只看见支持你当前信念的证据，
而忽略了那些挑战你假设的信号。
```

### 3. 因果转化链
**特点:**
- ✅ 解释过去行为如何导致现在
- ✅ 现在选择如何塑造未来
- ✅ 提供两条路径对比

**示例输出:**
```
因为你一直在等待"完美时机"，所以现在你发现自己困在原地，
既焦虑又动弹不得。这不是惩罚，而是能量守恒。

在接下来的72小时内，你需要做一个决定：
- 选项A（惯性路径）：继续当前模式 → 3个月后你会发现自己还在同一个地方打转
- 选项B（转化路径）：停止"准备"，开始一个不完美的行动 → 未必完美，但更真实
```

### 4. 精确到小时的建议
**特点:**
- ✅ 不说"放慢节奏"这种废话
- ✅ 给出具体时间和具体行动
- ✅ 包含禁止清单

**示例输出:**
```
### 今天下午（接下来3小时内）
整理出关于这件事的三个最核心问题，写在纸上。
不要答案，只要问题。在下午3点前完成。

### 明天的关键时段
上午9-11点：在笔记本上写下一个你一直拖延的具体行动
（必须是1小时内能完成的小事），然后在11点前完成它。

### 禁止清单（本周内避免）
❌ 向任何人寻求"你觉得我应该怎么办？"这种问题
❌ 查看任何形式的占卜、星座、运势
❌ 做任何"证明自己"的事
```

## 🎨 风格对比

### 温和模式（旧版）
- 语气：鼓励、温柔、不评判
- 建议：抽象、宽泛、留有余地
- 适合：需要情感支持的用户

### 教练模式（新版）
- 语气：犀利、直接、一针见血
- 建议：具体、精确、无可回避
- 适合：需要被推动改变的用户

## 📊 核心规则（强制执行）

1. **问题挂钩** - 每段分析首句必须回应问题关键词
2. **禁止引用** - 严禁"根据牌义"、"小阿卡纳"等说明性文字

4. **心理投射** - 使用心理学词汇增加专业深度
5. **拒绝模糊** - 不说"可能""或许"，用肯定语气
