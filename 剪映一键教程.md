# 🎬 剪映一键成片教程

## 下载剪映

**Mac版：** https://www.capcut.cn/  
**手机版：** App Store搜索"剪映"

---

## 一键操作流程（3分钟）

### 1. 打开剪映
- 点击「开始创作」

### 2. 导入视频
- 拖入你的 `螢幕錄影 2026-03-08 14.58.03.mov`

### 3. 一键成片（重点！）
```
点击顶部菜单「工具」→「智能成片」

或者

点击「图文成片」→ 输入文案 → AI自动生成
```

### 4. 选择风格
- 选择「清新」或「治愈」风格
- AI自动配乐、字幕、转场

### 5. 一键导出
- 点击「导出」
- 选择1080P
- 勾选「适配抖音」

**完成！**

---

## 💡 进阶技巧

### 智能字幕（自动生成）
```
1. 选中视频
2. 点击「文本」→「智能字幕」
3. 选择「中文」
4. 点击「开始识别」
5. 自动生成字幕！
```

### 一键卡点
```
1. 导入音乐
2. 点击「卡点」
3. AI自动匹配节奏
4. 完成！
```

### 智能抠图
```
1. 选中视频
2. 点击「智能抠图」
3. AI自动去背景
4. 添加新背景
```
