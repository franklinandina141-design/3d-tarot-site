# 🎬 剪映精修指南
## 基于已剪辑好的视频添加自然元素和特效

**基础视频：** `tarot_with_music.mp4` (23秒，已配音乐)  
**目标：** 清新治愈风 + 自然元素 + 专业特效

---

## 📱 第一步：导入素材

### 1. 打开剪映
- Mac：打开剪映专业版
- 手机：打开剪映App

### 2. 创建项目
```
1. 点击「开始创作」
2. 比例选择「9:16」（竖屏）
3. 导入视频：tarot_with_music.mp4
```

### 3. 下载自然元素素材（免费）

**推荐素材网站：**
- Pixabay: https://pixabay.com/
- Pexels: https://www.pexels.com/
- 剪映素材库（内置）

**需要的素材：**
- ✅ 樱花花瓣 PNG（透明背景）
- ✅ 白色羽毛 PNG
- ✅ 光点粒子 / Bokeh效果
- ✅ 云雾流动视频

---

## 🌸 第二步：添加自然元素

### 场景1：飘落樱花（0-8秒）

**操作步骤：**
```
1. 点击「画中画」→「新增画中画」
2. 选择樱花花瓣素材
3. 调整参数：
   - 大小：缩小到屏幕的 1/15
   - 位置：画面右上角
   - 不透明度：30%
4. 添加关键帧动画：
   - 0秒：位置在屏幕右上角外
   - 8秒：位置在屏幕左下角外
   - 旋转：0° → 360°（缓慢旋转）
5. 复制这个图层，调整位置，创建3-5片花瓣
```

**剪映快捷操作：**
- 选中花瓣图层 → 右键 → 复制
- 调整每片花瓣的起始位置和速度（错开时间）

### 场景2：光点粒子（8-20秒）

**操作步骤：**
```
1. 点击「特效」→「视频特效」
2. 搜索「光效」或「粒子」
3. 推荐特效：
   - 「光斑漂浮」
   - 「星光闪烁」
   - 「仙气飘飘」
4. 将特效拖到时间轴 8-20秒位置
5. 调整参数：
   - 不透明度：40%
   - 混合模式：滤色（Screen）
```

### 场景3：云雾流动（12-23秒）

**操作步骤：**
```
1. 导入云雾视频素材（或使用剪映内置）
2. 「画中画」→ 添加云雾素材
3. 调整：
   - 铺满全屏
   - 不透明度：15-20%
   - 混合模式：叠加（Overlay）
4. 速度：调整为 0.5x（更慢更梦幻）
```

## 📝 第三步：添加文字标题

### 开场标题（0-3秒）

**文案：** 「当科技遇见玄学...」

**操作步骤：**
```
1. 点击「文本」→「新建文本」
2. 输入文案，选择字体：
   - 推荐：霞鹜文楷 / 方正清刻本悦宋
   - 大小：48
   - 颜色：#2C4A3E（深墨绿）
3. 位置：居中偏上
4. 添加描边：
   - 颜色：白色
   - 宽度：2px
5. 添加动画：
   - 入场动画：「淡入上浮」
   - 持续时间：0.5秒
   - 出场动画：「淡出」（2.5秒时）
```

### 中间标题（8-10秒）

**文案：** 「不只是算命」

**操作步骤：**
```
1. 同上创建文本
2. 字体稍小：36
3. 位置：居中
4. 入场动画：「打字机效果」
```

### 中间标题2（10-13秒）

**文案：** 「而是一场深度的心理对话」

**操作步骤：**
```
1. 创建文本
2. 字体：32
3. 颜色：#9d85c4（淡紫色）
4. 入场动画：「渐显」
```

### 结尾标题（20-23秒）

**文案：** 「决定权，始终在你手中 ✨」

**操作步骤：**
```
1. 创建文本
2. 字体：40，加粗
3. 颜色：#F7E7CE（暖杏色）

```

## 🎨 第四步：调色和滤镜

### 整体调色

**操作步骤：**
```
1. 选中视频主轨道
2. 点击「调节」→「基础调节」
3. 参数设置：
   - 亮度：+5
   - 对比度：-3
   - 饱和度：-8（降低，更柔和）
   - 锐化：-5（柔化）
   - 色温：+300（偏暖）
4. 点击「高级调节」：
   - 高光：+15
   - 阴影：-10
```

### 添加滤镜

**推荐滤镜（剪映内置）：**
```
1. 点击「滤镜」
2. 选择类别「清新」或「氛围」
3. 推荐滤镜：
   - 「日系」系列：温柔柔和
   - 「莫兰迪」系列：高级感
   - 「白日梦」：梦幻感
4. 滤镜强度：30-40%（不要太重）
```

### 3. 音频混音

**操作步骤：**
```
1. 点击每条音频轨道
2. 调整音量平衡：
   - 背景音乐：60-70%
   - 音效：20-30%
3. 添加淡入淡出：
   - 开头：音乐淡入 0.5秒
   - 结尾：音乐淡出 1秒
```

## 💡 专业技巧

### 技巧1：图层管理
```
重命名图层：
- 花瓣1、花瓣2、花瓣3
- 文字-开场、文字-中间
- 避免混乱
```

### 技巧2：关键帧技巧
```
自然元素移动：
- 使用「缓动」曲线
- 避免匀速运动（不自然）
- 选择「ease-in-out」
```

### 技巧3：色彩一致性
```
确保所有添加的元素：
- 色调一致（暖色调）
- 饱和度接近
- 避免突兀的高饱和色彩
```

### 技巧4：不要过度
```
记住：少即是多
- 特效不要超过3种
- 文字不要超过5处
- 保持清新简洁
```
