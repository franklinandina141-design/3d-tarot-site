## 🧪 请执行这3个检查

### 检查1：浏览器控制台（最重要！）

**步骤：**
1. 按F12（或右键→检查）
2. 切换到"Console"标签
3. 刷新页面（Cmd+R）
4. **截图所有红色错误信息**

**常见错误及含义：**
```
❌ "Failed to fetch" → 数据文件加载失败
❌ "React is not defined" → React库未加载
❌ "Unexpected token" → JavaScript语法错误
❌ "Cannot read property" → 数据格式问题
```

### 检查2：Network标签

**步骤：**
1. F12 → Network标签
2. 刷新页面
3. 查看所有文件是否200 OK（绿色）
4. 特别检查：
   - `tarot-deck.local.json` 是否200？
   - `react.production.min.js` 是否200？
   - `mediapipe/hands` 是否200？

### 检查3：完全清除缓存

**Chrome：**
```
1. 设置 → 隐私和安全
2. 清除浏览数据
3. 时间范围：全部
4. 勾选：缓存的图片和文件
5. 清除数据
6. 重启浏览器
```

**或使用无痕模式：**
```
Cmd + Shift + N
在无痕窗口打开网址
```

## 💡 快速诊断脚本

**在浏览器Console中运行：**

```javascript
// 检查React
console.log('React:', typeof React !== 'undefined' ? '✅' : '❌');

// 检查数据
fetch('./assets/tarot-deck.local.json')
  .then(r => {
    console.log('数据HTTP:', r.ok ? '✅' : '❌');
    return r.json();
  })
  .then(d => {
    console.log('数据解析:', d ? '✅' : '❌');
    console.log('卡牌数:', d.cards ? d.cards.length : 0);
    console.log('第一张牌:', d.cards[0]);
  })
  .catch(e => console.error('❌ 错误:', e));
```
