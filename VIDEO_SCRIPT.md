# 🎬 塔羅網站演示視頻腳本

## 📊 視頻規格建議
- **時長：** 60-90秒（社交媒體黃金時長）
- **尺寸：** 9:16 豎屏（抖音/小紅書/Instagram Reels）或 16:9 橫屏（YouTube/B站）
- **語氣：** 輕鬆、真誠、略帶神秘感

---

## 🎭 腳本結構

### **開場（0-10秒）**

**畫面：** 網站封面頁 + 溫柔的背景音樂

**文案（字幕/旁白）：**
```
「花了一週時間，用 AI 做了一個塔羅網站」
「專業級解讀 × 心理占卜 × 零廢話」
```

**操作：**
- 鼠標懸停在標題上
- 展示問題輸入框

---

### **第一幕：展示輸入問題（10-20秒）**

**畫面：** 輸入問題的過程

**文案：**
```
「不只是算命，而是深度心理對話」
「我問：這段關係還值得投入嗎？」
```

**操作：**
- 慢慢打出問題：「這段關係還值得投入嗎？」
- 點擊「開啟靈魂儀式」按鈕

---

### **第二幕：手勢抽牌（20-35秒）**

**畫面：** 手勢識別抽牌過程

**文案：**
```
「用攝像頭捕捉手勢，像真的在抽牌」
「每張牌都有正位/逆位，隨機生成」
```

**操作：**
- 展示手在螢幕前揮動
- 捏合手指選牌的動作
- 三張牌飛入上方卡槽的動畫

**提示：** 如果手勢識別不穩定，可以用點擊模式錄製

---

### **第三幕：解讀結果亮點（35-70秒）**

**畫面：** 滾動展示五大板塊

**文案：**
```
「AI 整合了三本經典塔羅書籍」
「每個板塊都不是模板，是真正的深度分析」

板塊1：「靈魂氣候總覽」
→ 展示：「你正處於內向能量期...」

板塊2：「現狀深度解碼」  
→ 展示：「隱藏的焦慮、心理防禦、真實渴望」

板塊4：「你現在最需要的方向」
→ 展示：「心智轉變 + 具體行動 + 時機把握」

板塊5：「給你的溫柔結語」
→ 展示：「決定權始終在你手中」
```

**操作：**
- 慢速滾動，讓觀眾看清文字
- 停留在幾個關鍵句子上

---

### **第四幕：技術細節（70-80秒）**

**畫面：** 切到代碼編輯器/終端（可選）

**文案：**
```
「技術棧：React + 手勢識別 + AI 提示工程」
「完全開源，代碼在 GitHub（可選）」
```

**操作：**
- 快速展示一段代碼（看起來專業）
- 或展示項目文件夾

---

### **結尾（80-90秒）**

**畫面：** 回到網站首頁

**文案：**
```
「塔羅不是算命，是一面溫柔的鏡子」
「AI 也可以很有靈性 🔮」

「點贊收藏，我把項目鏈接放評論區了」
```

**操作：**
- 淡出到黑色
- 顯示你的社交媒體賬號

## 🎥 錄製建議

### **工具選擇**

**Mac 原生：**
- **QuickTime Player** - 免費，錄螢幕 + 攝像頭
- **Cmd + Shift + 5** - 系統自帶螢幕錄製

**專業工具：**
- **OBS Studio** - 免費，功能強大，可同時錄螢幕和攝像頭
- **ScreenFlow** - 付費，Mac 專用，編輯友好

**我的建議：** 先用系統自帶錄製，簡單快速

---

### **錄製清單**

#### **鏡頭1：開場**
- 錄製網站首頁 5-10秒
- 保持靜止，方便加字幕

#### **鏡頭2：輸入問題**
- 錄製打字過程（可以加速）
- 點擊按鈕的過渡動畫

#### **鏡頭3：抽牌過程**
- 錄製手勢識別（如果不順利，用點擊也行）
- 抓住卡牌飛入的動畫瞬間

#### **鏡頭4：滾動解讀**
- 慢速滾動 5 個板塊
- 每個板塊停留 3-5 秒

#### **鏡頭5：結尾**
- 回到首頁，錄製 3 秒

## ✂️ 後期編輯方案

### **方案A：簡單剪輯（推薦）**

**工具：** iMovie（Mac 免費自帶）

**步驟：**
1. 導入所有錄製素材
2. 裁剪掉冗餘部分
3. 添加字幕（手動或自動）
4. 添加背景音樂（推薦：輕柔的 Lo-fi 或氛圍音樂）
5. 導出為 1080p MP4

**我可以幫你：**
- 生成字幕文件（.srt）
- 用 ffmpeg 批量處理視頻（剪輯、加速、添加字幕）

**我可以幫你：**
