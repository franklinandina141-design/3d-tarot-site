## 📽️ 分镜脚本（25秒版本）

### 【0-3秒】开场Hook - "治愈系神秘"

**画面：**
- 0.0s: 黑屏淡入，出现一朵缓缓绽放的白色花朵（CG特效）
- 0.5s: 花朵转化为发光的塔罗牌轮廓
- 1.0s: 切到你的网站首页，标题缓缓浮现

**文案（字幕）：**
```
「当科技遇见玄学...」
或
「AI 也能读懂你的灵魂？」
```

**转场方式：** 
- 光晕扩散转场（Lens Flare Transition）
- 配合轻柔的"叮"声效

**配乐：**
- 开场用水晶钵音效（Crystal Singing Bowl）
- 底层铺垫柔和的氛围音乐

---

### 【3-8秒】展示操作 - "交互美学"

**画面：**
- 3s: 手指轻触屏幕，输入问题（慢动作 0.8x）
- 5s: 点击"开启灵魂仪式"按钮，按钮发光特效
- 6s: 快切到手势抽牌画面（加速 1.5x）
- 7s: 三张牌飞入卡槽的动画（保持原速）

**文案（字幕）：**
```
3s: 「像和朋友聊天一样简单」
6s: 「用手势，选出属于你的牌」
```

**转场方式：**
- 擦除转场（Wipe Transition）- 柔和的从左到右
- 或使用"水波纹扩散"转场

**自然元素叠加：**
- 在画面边缘添加飘落的樱花花瓣（透明度30%）
- 或柔和的光斑粒子（Bokeh效果）

---

### 【8-15秒】核心卖点 - "深度解读"

**画面：**
- 8s: 切到解读结果页面
- 9s: 缓缓向下滚动（慢动作 0.7x）
- 10s: 停在"灵魂气候总览"板块，高亮显示
- 12s: 快速滚动经过"现状深度解码"
- 13s: 停在"给你的温柔结语"，定格

**文案（字幕）：**
```
8s:  「不只是算命」
10s: 「而是一场深度的心理对话」
13s: 「决定权，始终在你手中 ✨」
```

**转场方式：**
- 滚动时使用"柔焦模糊"效果（Soft Focus Blur）
- 定格时用"光晕聚焦"（Vignette Focus）

**自然元素叠加：**
- 文字出现时配合"晨露滴落"效果
- 背景添加柔和的云雾流动（透明度20%）

---

### 【15-20秒】技术展示 - "科技美学"

**画面：**
- 15s: 分屏展示（三宫格）
  - 左：网站首页
  - 中：手势识别抽牌
  - 右：解读结果滚动
- 18s: 三宫格合并为完整画面

**文案（字幕）：**
```
15s: 「AI × 心理学 × 传统智慧」
17s: 「每一次解读，都是独一无二」
```

**转场方式：**
- 三宫格用"渐隐叠加"（Crossfade Dissolve）
- 合并时用"向心聚拢"（Center Squeeze）

**自然元素叠加：**
- 分屏边框用藤蔓/树枝装饰（简约线条）
- 背景添加星空粒子缓缓移动

---

### 【20-25秒】结尾CTA - "治愈召唤"

**画面：**
- 20s: 淡出到半透明黑底
- 21s: 显示网站Logo或二维码（居中）
- 22s: 周围浮现发光的塔罗牌符号（缓慢旋转）
- 24s: 淡入文案

**文案（字幕）：**
```
20s: 「今天，听听你的内在声音」
23s: 「链接在评论区 🔮」
```

**转场方式：**
- 淡入淡出（Simple Fade）
- 配合柔和的光晕扩散

**自然元素叠加：**
- 周围飘落柔和的羽毛/光点
- 背景用极简的星空或薄雾

**音效：**
- 结尾用风铃声（Wind Chimes）
- 配合柔和的"嗡"声余韵

## 🎨 视觉设计系统

### 色彩方案（清新治愈风）

**主色调：**
```
- 奶油白：#F7F3E9（背景底色）
- 薄荷绿：#B8E6D5（强调色）
- 暖杏色：#F7E7CE（文字主色）
- 淡紫色：#D4C5E2（点缀色）
- 深墨绿：#2C4A3E（对比色）
```

**调色参数（达芬奇/PR）：**
```
- 高光：+15（柔化）
- 阴影：-10（提亮）
- 饱和度：-8（降低，更柔和）
- 色温：+300K（偏暖）
- 添加柔光滤镜（Soft Glow）强度20%
```

### 字体设计

**主标题：**
- 字体：霞鹜文楷 / 方正清刻本悦宋
- 大小：48px
- 颜色：#2C4A3E（深墨绿）
- 效果：添加柔和白色描边（2px）+ 淡淡投影

**副标题/说明文字：**
- 字体：思源黑体 Light
- 大小：32px
- 颜色：#6B7B75（灰绿色）
- 效果：无描边，仅淡淡投影

**出字动画：**
- 淡入 + 轻微上浮（20px）
- 持续时间：0.4秒
- 缓动函数：ease-out

## 🌿 自然元素叠加库

### 可用素材（免费来源）

**飘落元素：**
1. 樱花花瓣（透明PNG）
   - 来源：Pexels / Pixabay

   - 颜色：白色/奶油色

3. 光点粒子（Bokeh）
   - 颜色：暖白/薄荷绿/淡紫
   - 数量：20-30个
   - 大小：不规则（10-30px）
   - 运动：缓慢浮动

**背景元素：**
1. 云雾流动
   - 透明度：15-20%
   - 速度：极慢
   - 颜色：淡紫/薄荷绿渐变

2. 星空粒子
   - 用于结尾（20-25秒）
   - 透明度：40%
   - 运动：闪烁 + 缓慢移动

3. 藤蔓/树枝边框
   - 用于三宫格（15-20秒）
   - 风格：极简线条

```

+ 展示操作（5秒）
+ 核心卖点（7秒）

```
惊艳（0-3s）→ 好奇（3-8s）→ 向往（8-15s）→ 信任（15-20s）→ 行动（20-25s）

```
快节奏（3-8秒）：展示功能，保持刺激

```
